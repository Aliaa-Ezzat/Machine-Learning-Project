## ✨ **Features**

| Feature | Description |
|---------|-------------|
| 📊 **Dataset Management** | Upload CSV files, view data tables, and explore dataset info |
| 📈 **Visualization** | Interactive histograms, scatter plots, and correlation matrices |
| 🛠️ **Preprocessing** | Handle missing values, encode categorical data, and scale features |
| 🤖 **Model Training** | Train multiple ML models with hyperparameter tuning |
| 📉 **Evaluation** | View accuracy, precision, recall, and confusion matrices |
| 💾 **Export Results** | Save processed data and model outputs as CSV |
